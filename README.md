A LinkedIn-style social media application built using the MERN Stack with real-time features and modern full-stack architecture.
This project includes:

Secure User Authentication using JWT & bcrypt.js

Post Creation with text and image upload using Multer + Cloudinary

Real-Time Likes, Comments & Notifications powered by Socket.io

Connection System (send, accept, reject requests)

User Profiles with editable details & profile picture

Powerful Search for users and posts

Global State Management using React Context API

This project is designed to help developers learn complete, real-world MERN development, including authentication, real-time updates, cloud image storage, and clean frontend/backend structure.
